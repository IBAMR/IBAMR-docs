Detailed Description

template<int DIM>
class SAMRAI::appu::EmbeddedBoundaryShapeSphere< DIM >

The user must specify in the input a "center" and a "radius". An example input entry would look like:

*        Shape1 {
*           type = "SPHERE"
*           center = 40.0 , 15.0, 15.0
*           radius = 5.0
*        }
