Detailed Description

template<int DIM, class TYPE>
class SAMRAI::pdat::EdgeDataFactory< DIM, TYPE >

Class EdgeDataFactory is a factory class used to allocate new instances of EdgeData objects. It is a subclass of the patch data factory class and edge data is a subclass of patch data. Both the factory and data classes are templated on the type of the contained object (e.g., double or int).
