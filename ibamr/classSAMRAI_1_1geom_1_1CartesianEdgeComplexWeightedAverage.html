Detailed Description

template<int DIM>
class SAMRAI::geom::CartesianEdgeComplexWeightedAverage< DIM >

Class CartesianEdgeComplexWeightedAverage implements conservative edge-weighted averaging for edge-centered complex patch data defined over a Cartesian mesh. It is derived from the xfer::CoarsenOperator<DIM> base class. The numerical operations for theaveraging use FORTRAN numerical routines.

The findCoarsenOperator() operator function returns true if the input variable is edge-centered complex, and the string is "CONSERVATIVE_COARSEN".

Coarsen the source component on the fine patch to the destination component on the coarse patch using the edge-centered complex weighted averaging operator. Coarsening is performed on the intersection of the destination patch and the coarse box. It is assumed that the fine patch contains sufficient data for the stencil width of the coarsening operator.

Implements SAMRAI::xfer::CoarsenOperator< DIM >.
