IBAMR::CIBFEMethod Class Reference

Class CIBFEMethod is a concrete CIBStrategy and IBFEMethod class which implements the motion of rigid bodies using the constraint formulation. The immersed structure is discretized using finite-element mesh. More...

Detailed Description

Class CIBFEMethod is a concrete CIBStrategy and IBFEMethod class which implements the motion of rigid bodies using the constraint formulation. The immersed structure is discretized using finite-element mesh.
