Detailed Description

Class KrylovLinearSolverPoissonSolverInterface provides an interface for KrylovLinearSolvers that are to be used as Poisson solvers.

This class is intented to be used to create a (trivial) subclass of an existing implementation of KrylovLinearSolver that also supports the PoissonSolver interface.

See also
PETScKrylovPoissonSolver
