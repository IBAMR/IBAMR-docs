Detailed Description

template<int DIM>
class SAMRAI::pdat::FirstLayerCellFillPattern< DIM >

Class FirstLayerCellFillPattern is a concrete implementation of the abstract base class VariableFillPattern. It is used to calculate overlaps according to a pattern which limits overlaps to the cell-centered ghost region of width 1 surrounding a patch, including all edges and corners.

Calculate overlaps between the destination and source geometries according to the desired pattern. This will return the portion of the intersection of the geometries that lies in the ghost region of width 1 surrounding the patch. The patch is identified by the argument dst_patch_box.
