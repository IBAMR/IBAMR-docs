Detailed Description

template<int DIM>
class SAMRAI::xfer::FillBoxSet< DIM >

Class FillBoxSet is a utility class that provides a "smart" set of boxlist operations for communication schedules. Specifically, it provides some of the basic box calculus operations, such as set intersection and difference, found in the simple boxlist class. However, this class is used in very specific situations involving "fill boxes" in communication schedule creation, to avoid potentially expensive exhaustive searches. Essentially, it limits searches for box intersections to those that intersect the bounding box.
